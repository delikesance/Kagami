# Kagami - Design Guide (Gruvbox Dark)

## Vision
**Retro-Groove & Warmth**. Kagami adopte une esthétique "vintage tech". C'est chaleureux, contrasté mais doux pour les yeux, rappelant les terminaux haut de gamme des années 80.

## Palette de Couleurs (Gruvbox)
- **Background (bg0)**: `#282828` - Fond principal.
- **Surface (bg1)**: `#3c3836` - Éléments secondaires.
- **Foreground (fg)**: `#ebdbb2` - Texte principal (crème).
- **Gray**: `#928374` - Sous-texte.
- **Accent (Aqua)**: `#8ec07c` - Couleur primaire (Information).
- **Success (Green)**: `#b8bb26`
- **Warning (Yellow)**: `#fabd2f`
- **Error (Red)**: `#fb4934`
- **Orange**: `#fe8019` - Utilisation pour le prestige/niveaux.

## Principes UI
1. **Soft Contrast**: Éviter le noir pur et le blanc pur.
2. **Sharp Geometry**: Bordures nettes mais couleurs chaudes.
3. **Hierarchy**: Utiliser les variantes de gris Gruvbox pour la profondeur.
4. **Retro-Modern**: Typographie claire sur fonds terreux.
